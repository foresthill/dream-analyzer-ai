# Dream Analyzer Mobile App

This directory is reserved for the future mobile implementation using Expo (React Native).

## Planned Stack
- **Framework**: Expo SDK 52+
- **Navigation**: Expo Router
- **UI**: React Native Paper or NativeWind

## Setup (Future)

```bash
npx create-expo-app@latest . --template expo-router
```

The mobile app will share:
- `@dream-analyzer/shared-types` - Type definitions
- `@dream-analyzer/dream-core` - Business logic and AI analysis
- `@dream-analyzer/ui-components` - Cross-platform UI components
