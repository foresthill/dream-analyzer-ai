# シンボル解釈ガイド

## シンボルカテゴリ

### 人物 (people)
- 自己の側面
- 重要な人間関係
- 内なる声

### 動物 (animals)
- 本能的な側面
- 特定の性質や特性
- 自然との繋がり

### 場所 (places)
- 心理状態
- 人生の段階
- 安全性や危険

### 物体 (objects)
- ツールや資源
- 記憶や関連
- 象徴的な意味

### 行動 (actions)
- 対処方法
- 欲求や恐怖
- 変化のプロセス

### 自然 (nature)
- 感情の状態
- 生命力
- 変化と成長

## 解釈のガイドライン

1. **コンテキストを考慮する**
   夢全体の文脈でシンボルを解釈する

2. **個人的な関連を重視する**
   一般的な意味より個人的な経験を優先

3. **感情を重視する**
   シンボルに対する感情的反応を考慮

4. **複数の解釈を検討する**
   一つの解釈に固執しない
