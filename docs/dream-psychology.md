# 夢分析の心理学的背景

## 主要な理論

### フロイトの夢分析
- 夢は無意識の願望の表れ
- 顕在内容と潜在内容
- 象徴的表現の解釈

### ユングの分析心理学
- 集合的無意識
- 元型（アーキタイプ）
- 個性化のプロセス

### 現代の認知アプローチ
- 記憶の統合
- 感情の処理
- 問題解決

## 夢の要素

### 感情
夢で感じた感情は、起きている時の感情状態を反映していることが多い。

### シンボル
普遍的なシンボルと個人的なシンボルの両方が存在する。

### 物語構造
夢のストーリーは、内面の葛藤や願望を表現することがある。

## 注意事項

- 夢の解釈は主観的であり、絶対的な正解はない
- 専門的な心理サポートが必要な場合は専門家に相談すること
- このアプリの分析は参考情報であり、医療・心理診断ではない
